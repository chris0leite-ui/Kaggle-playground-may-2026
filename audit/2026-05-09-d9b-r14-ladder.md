# Day-9b — R14 strength ladder

> Day-9 closed: R14 hash_lr_3way was the only base near the min-meta
> gate (Δ −0.02bp, ρ vs PRIMARY 0.444). Hypothesis: R14's LR class
> has feature-engineering headroom — strengthening features raises std
> OOF without collapsing diversity, lifting the gate to PASS. Builders
> `scripts/d9b_r14_ladder.py` + `scripts/d9b_kn_stack.py`.

## Ladder definitions

Each level adds to the previous; all use 2¹⁸ hashed features +

## Key insight — strength × diversity tradeoff

L0 → L1 added Race + Year + their 2-way interactions. Std OOF jumped
+10bp (0.79 → 0.89), but ρ vs PRIMARY also jumped (0.44 → 0.84) —
the LR is now duplicating signals that GBDT bases already extract.
**Net min-meta got *worse*** (−0.02 → −0.06 bp).

L1 → L2 added 5-quintile bins of TyreLife, RaceProgress, Position as
main effects (no high-order interactions with Race/Year/Driver). Std
OOF +2bp; ρ +0.03; **min-meta flipped from −0.06 to +0.01 (PASS)**.

The interpretation: GBDT trees naturally split numerics. The LR can
only express piecewise-constant patterns by binning them as
categoricals first. Once the LR has those bin features, it produces
predictions that are still mostly redundant with GBDT but with a
small *additive-log-odds residual* the meta can route — exactly the
~0.5bp signal the F1.2 rule_residuals each contribute.

L4 → L5 confirms the limit: 3-way Driver × Compound × numeric
interactions overfit liblinear LR even at C=1 with the 2¹⁸ hash
budget; std OOF and min-meta both regress.

## Triage decision

1. **Submission EV remains marginal**: predicted Δ LB is +0.19bp,
   below the +0.5bp slot-worthy threshold from F1.2's predicted_lb
   heuristic. Per CLAUDE.md Rule 1, no submission without PI sign-off
   — and the EV is below the gap-tracking floor.
2. **L4 swap stack is the new local optimum** for the rule_residual
   + LR-augmented mechanism family. Gain over Day-9 S4 = +0.07bp OOF.
3. **L2 is structurally cleanest** (no Driver-numeric overfit): nearly
   identical OOF (+0.18 vs L4's +0.19) at higher diversity in L1
   ranking. If submitting, L2 has a slight ρ-safety margin.
4. **The ladder confirms the Day-9 pattern**: model-class diversity is
   real but small once the LR sees the same features the GBDTs do.
   Lift requires either truly orthogonal signals (external data) or a
   structurally different loss (hazard NN, factorization machine).
